# Getting Started

1. Clone the repository

```
> git clone https://github.com/dhimanroyit/travelers_client.git
```

2. Change Directory

```
> cd travelers_client
```

3. Install dependency

```
> npm install
```

4. Run the app

```
> npm start
```

Done
